# Sangeetha Mobiles Shop Inventory Management System

## BCA Final Year Project

### Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

### Modules
- Admin Management
- Product Management
- Supplier Management
- Stock Management
- Customer Management
- Billing Management

### Project Description
This project is developed to manage mobile inventory, stock, suppliers, customers, sales, and billing operations efficiently for Sangeetha Mobiles Shop.

### Features
- Add, Update and Delete Products
- Manage Stock Inventory
- Supplier Management
- Customer Management
- Billing System
- Sales Report Generation
